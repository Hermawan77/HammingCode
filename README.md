# HammingCode

Contoh Eksekusi
Input :
Binner Ke-1 : 1
Binner Ke-2 : 0
Binner Ke-3 : 1
Binner Ke-4 : 1
Binner Ke-5 : 1
Binner Ke-6 : 0
Binner Ke-7 : 0

Output :
Pesan : 1 0 1 1 1 0 0
A salah
A diubah menjadi : 0
Pesan dan Parity yang benar : 0 0 1 1 1 0 0
